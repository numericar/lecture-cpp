Variable
	เป็นชื่อที่เราตั้งให้กับข้อมูลที่เราต้องการจะเก็บ 
	โดยเป็นแนวคิดที่เราได้มาจากคณิตศาสตร์ เช่น  f(x) = x^2 + 4x

	int a = 10;
	char b = 'A';
	float c = 12.75f; (หากเลขมีทศนิยม การระบุ f จะทำให้ compiler รู้ว่าข้อมูลนั้นคือ float ถ้าไม่ระบุจะถูกมองเป็น double)

	การตั้งชื่อตัวแปร ห้ามขึ้นต้นด้วยตัวเลข แต่สามารถมีตัวเลขในชื่อตัวแปรได้
	เช่น
		- ผิด:	1x
		- ถูก:	x1
