Data type in C++
	โปรแกรมจะประกอบไปด้วย data และ instruction
	โดย instruction คือ กระบวนการทำงาน ที่จะนำมาจัดการกับ data ที่โปรแกรมมี

	โดยทุดส่วนของโปรแกรมจะถูกเก็บไว้่ใน memory
		heap			[]
		stack			[data]
		code section	[instruction]

	stack 
		จัดเก็บตัวแปรทั้งหมดที่มีอยู่ในโปรแกรม
	code section
		จัดเก็บชุดคำสั่งทั้งหมดของโปรแกรม

	โดยชนิดของข้อมูลจะถูกแบ่งออกเป็นดังนี้
		1. numeric เป็นข้อมูลที่เกี่ยวข้องกับตัวเลข
			1.1	integer เป็นตัวเลขที่ไม่มีทศนิยม
			1.2 floating เป็นตัวเลขที่มีทศนิยม
		2. character เป็นตัวอักษร 1 ตัว
		3. string เป็นกลุ่มของ character ที่มารวมกันกลายเป็นคำ

	** 1 byte = 8 bits
	โดยขนาดของ byte จะมีผลกับขนาดของข้อมูล ยิ่้งจำนวน byte เยอะ ก็จะยิ่งเก็บข้อมูลได้ขนาดใหญ่ขึ้น
