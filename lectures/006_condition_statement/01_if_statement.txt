If statement
    Conditional statement   
        เป็นการประมวลผลตัดสินใจ ตามสถานการณ์ที่เกิดขึ้น
    
    if 
        เป็นการตัดสินใจว่าจะดำเนินการอะไร ตามเงื่อนไขที่กำหนด 
        ตัวอย่างชุดคำสั่ง
            if (condition) 
            {

            }
            else 
            {

            }

        โดยถ้าหาก condition เป็นจริง จะดำเนินการคำสั่งที่อยู่ใน {} ของ if 
        แต่ถ้าเป็นเท็ต จะดำเนินการคำสั่งที่อยู่ใน {} ของ else

        โดย condition จะใช้ relational operator (นิพจน์เชิงความสัมพันธ์) นิพจน์เชิงความสัมพันธ์ ประกอบด้วย 
            1. <
            2. <=
            3. >
            4. >=
            5. ==
            6. !=
        ตัวอย่าง เช่น 
            int a = 5, b = 10;
            bool c = a < b;
            ค่าใน c จะเป็นจริง เพราะ a น้อยกว่า b
