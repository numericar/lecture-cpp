Skeleton of C++ Program
    ทุกโปรแกรมของ c++ จะเริ่มต้นที่ function main()
    และจะต้อง return int เสมอ

    int main() {
        return 0;
    }

    cout หมายถึง console out จะเป็นการแสดงผลข้อมูลไปที่ console terminal

    << หมายถึง insertion operator

    Library หมายถึง object หรือ function ที่โปรแกรมเมอร์สามารถนำมาใช้ในโปรแกรมของตัวเองได้

    iostream หมายถึง Library สำหรับจัดการ input และ output

    :: หมายถึง scope resolution operator ใช้สำหรับเรียกใช้งาน function 

    using namespace <namespace_name> ทำให้เราสามารถเรียกใช้งาน function, variable ของ namespace นั้นได้เลย ไม่ต้องใช้ scope resolution operator
