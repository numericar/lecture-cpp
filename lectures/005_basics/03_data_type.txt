Data type in C++
	โปรแกรมจะประกอบไปด้วย data และ instruction
	โดย instruction คือ กระบวนการทำงาน ที่จะนำมาจัดการกับ data ที่โปรแกรมมี

	โดยทุดส่วนของโปรแกรมจะถูกเก็บไว้่ใน memory
		heap			[]
		stack			[data]
		code section	[instruction]

	stack 
		จัดเก็บตัวแปรทั้งหมดที่มีอยู่ในโปรแกรม
	code section
		จัดเก็บชุดคำสั่งทั้งหมดของโปรแกรม

	โดยชนิดของข้อมูลจะถูกแบ่งออกเป็นดังนี้
		1. numeric เป็นข้อมูลที่เกี่ยวข้องกับตัวเลข
			1.1	integer เป็นตัวเลขที่ไม่มีทศนิยม
			1.2 floating เป็นตัวเลขที่มีทศนิยม
		2. character เป็นตัวอักษร 1 ตัว
		3. string เป็นกลุ่มของ character ที่มารวมกันกลายเป็นคำ

	** 1 byte = 8 bits
	โดยขนาดของ byte จะมีผลกับขนาดของข้อมูล ยิ่้งจำนวน byte เยอะ ก็จะยิ่งเก็บข้อมูลได้ขนาดใหญ่ขึ้น

	ชนิดของข้อมูลจะประกอบไปด้วย
		1. Primitive 
		2. User Defined
		3. Derived 

	Primitive Data Type
		เป็นชนิดข้อมูลดั้งเดิมที่มีอยู่ใน c++ โดยมีดังนี้
			1. integral หมายถึง ชนิดข้อมูลที่ไม่มีทศนิยม
				1.1 int 
				1.2 char
			2. bool หมายถึง ชนิดข้อมูลที่ประกอบด้วย จริง หรือ เท็จ
			3. floating point หมายถึง ชนิดข้อมูลที่มีทศนิยม
				3.1 float
				3.2 double

		โดยแต่ละชนิดข้อมูล primitive ตะมีขนาดดังนี้
		data_type		size
		int				2 or 4 (ขนาดจะขึ้นอยู่กับ version ของ compiler)
		float			4
		double			8
		char			1
		bool			undefined

		เราสามารถใช้สิ่งที่เรียกว่า modified เพื่อแก้ไขชนิดข้อมูลได้โดยประกอบด้วย
			1. unsigned ทำให้ชนิดตัวแปรที่เราระบุเก็บเฉพาะค่า + เท่านั้น เช่น unsigned int จากปกติจะเก็บ -32768 ~ 32767 ก็จะเก็บเป็น 0 ~ 65535
			2. long ทำให้ชนิดข้อมูล int สามารถเก็บข้อมูลได้ขนาดใหญ่ขึ้น เช่น long int จะทำให้ int จากปกติใช้ 2 byte เป็นใช้ได้ 4 byte แต่ถ้า int มีขนาด 4 byte อยู่แล้วจะทำให้กลายเป็น 8 byte
			   ถ้านำมาใช้กับ float จะทให้กลายเป็น 8 byte และถ้าใช้กัย double จะทำให้เป็น 10 byte

	User Defined
		เป็นชนิดข้อมูลที่ผู้ใช้สร้างขึ้นมาเอง โดยมีดังนี้
			1. enum
			2. strcture
			3. union
			4. class

	Derived
		1. Array
		2. Pointer
		3. Reference
