Operator and Expression
	เป็น function ใน c++ ที่ถูกสร้างขึ้นมาเพื่อดำเนินการกับอะไรบางอย่างเกี่ยวกับคณิตศาสตร์กับข้อมูล
	โดยใน c++ มี operator ดังนี้
		1. Arthimetic	เป็นตัวดำเนินการทางคณิตศาสตร์ ประกอบด้วย
			1.1 +	บวก
			1.2 -	ลบ
			1.3 *	คูณ (astrik)
			1.4 /	หาร
			1.5 %	หารเอาเศษ (mod) โดยสามารถใช้ได้เฉพาะ int และ char
		2. Relational	เป็นตัวดำเนินการที่ใช้เปรียบเทียบข้อมูลทั้ง 2 อย่าง ประกอบด้วย
			2.1 <
			2.2 <=
			2.3 >
			2.4 >=
			2.5 ==
		3. Logical		เป็นตัวดำเนินการใช้เปรียบเทียบว่าเป็น จริง หรือ เท็จ ประกอบด้วย
			3.1 &&	and
			3.2 ||	or
			3.3 !	not
		4. Bitwise		เป็นตัวดำเนินการที่ใช้ในการดำเนินการระดับ bit ของข้อมูล ประกอบด้วย
			4.1 &	and
			4.2 |	or
			4.3 ~	not
			4.4 !	xor
		5. Increment and Decrement	ใช้สำหรับเพิ่มหรือลบค่าของข้อมูล เช่น int x = 10; x--; x = 9;
		5. Assignment ใช้เพื่อกำหนดข้อมูลให้กับตัวแปร


	type casting
		เป็นการแปรงข้อมูลจากชนิดหนึ่ง เป็นอีกชนิดหนึ่ง
		เช่น ต้องการแปรง int เป็น float จะได้ float x = (float) 12;

	หารเกิดการหานแล้วมีเศษ 
	เช่น
		int a = 13, b = 5;
		float c;
		c = a / b;
	ผลลัพที่ได้ควรจะเป็น 2.6 แต่ผลลัพที่ได้จริงคือ 2 เนื่องจาก int จะไม่เก็บทศนิยม
	เราจะต้องทำการแปรงชนิดข้อมูลด้วย type casting
		c = (float) / b;

	Expression (นิพจน์)
		เป็นการผสมผสานกันระอว่างตัวแปร ค่าคงที่ ตัวดำเนินการ และฟังก์ชัน เพื่อทำงานร่วมกันให้ได้ผลลัพธ์ยางอย่าง 
		โดยสามารถเป็นได้หลายรูปแบบ เช่น การดำเนินการทางคณิตศาสตร์ การเปรียบเทียบเชิงตรรกะ หรือการเรีนกฟังก์ชัน
		เช่น 
			1. Arthimetic Expression (นิพจน์ทางคณิตศาสตร์)
			2. Logical Expression (นิพจน์ทางตรรกะ)
			3. Relational Expression (นิพจน์เชิงความสัมพันธ์)
		โดยมีลำดับความสำคัญของการดำเนินการดังนี้
			operator		assumed preccedence
			()				3
			*, /, %			2
			+, -			1
