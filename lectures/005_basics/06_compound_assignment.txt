Compound Assignment Operator
    ใช้เมื่อเราต้องการเพิ่มค่าอะไรบางอย่างให้กับตัวมันเอง
    เช่น เราต้องการให้ a มีค่าเพิ่มขึ้น 5 หน่วย 
    โดยปกติเราจะเขียนแบบนี้ a = a + 5; 
    แต่เราสามารถลดรูปมันได้โดยการใช้ a += 5; สิ่งนี้เรียกว่า Compound Assignment

    โดยประกอบด้วย
        += -> a = a + n = a += n
        -= -> a = a - n = a -= n
        *= -> a = a * n = a *= n
        /= -> a = a / n = a /= n
        %= -> a = a % n = a %= n
