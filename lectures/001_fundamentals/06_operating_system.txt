What is Operation System
    บนโลกของเรามีระบบปฏิบัติการที่ได้รับความนิยมอยู่ 5 ตัว คือ
        1. windows
        2. linux
        3. maxos
        4. android
        5. ios
    โดยระบบปฏิบัติการ คือ โปรแกรมหลักที่จะนำทรัพยากรณ์ทั้งหมดในคอมพิวเตอร์ของเรามาให้บิรการแก่ผู้ใช้
    โดยคอมพิวเตอร์มักจะประกอบไปด้วย
        1. cpu
        2. main memory
        3. keyboard
        4. mouse
        5. monitor
        6. harddisk
        6. network
    โดยระบบปฏิบัติการจะทำงานอยู่ใน main memory และจะถูกโหลดเข้าไปใน main memory โดยอัตโนมัติเมื่อมีการเปิดใช้งานคอมพิวเตอร์
    
    และเมื่อโปรแกรมอื่น ๆ ต้องการาสั่งงานไปที่อุปกรณ์ที่เป็น hardware มันจะต้องส่งคำร้องไปที่ระบบปฏิบัติ และระบบปฏิบัติการจะดำเนินการต่อให้
