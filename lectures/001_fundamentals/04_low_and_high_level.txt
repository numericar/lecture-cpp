Low-level and High-level language
    Low-level คือ ภาษาที่คอมพิวเตอร์สามารถเข้าใจได้ คือ binary หรือเรียกอีกชื่อได้ว่า machine language
    ** Low-level ประกอบด้วย 2 ภาษา คือ binary, assembly

    Assembly language
        เป็นภาษาคอมพิวเตอร์ที่เกิดขึ้นมาเพื่อทำให้การเขียนโปรแกรมง่ายขึ้น โดยสามารถเรียนรู้ได้ง่ายขึ้น อ่านเข้าใจได้ง่ายขึ้น

    ถึงแม้ว่าจะมีภาษา assembly แล้ว แต่ตัวภาษายังคงเรียนรู้ได้ยากอยู่ดี เราจึงต้องการภาษาที่มีความใกล้เคียงกับภาษาอังกฤษ
    ที่สามารถเรียนรู้ได้ง่าย จึงเกิด High-level ขึ้นมา 
    ตัวอย่างภาษาระดับสูง
        - c/c++
        - c#
        - java 
        - python
        - visual basic
